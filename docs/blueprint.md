# **App Name**: Secure File Hub

## Core Features:

- User Authentication: Secure user authentication with bcrypt for password hashing and JWT for session management.
- File Upload and Management: Allows users to upload files, classify them, and view their status.
- Admin Review and Approval: Enables admins to review files, provide feedback, and approve or reject submissions.
- Automated Acta Generation: Automatically generates acceptance actas in PDF format upon approval.
- Email Notifications: Sends email notifications to users regarding the status of their uploads and any required actions.
- Audit Logging: Maintains a log of all actions performed by users and administrators for security and compliance purposes.
- Dashboard Metrics: Provides admins with a dashboard displaying key metrics related to file submissions and approvals.

## Style Guidelines:

- Primary color: Deep institutional blue (#004A99) to convey trust and stability.
- Background color: Light, desaturated blue (#E5F0FA) for a clean and professional look.
- Accent color: Soft gold (#D4A76A) to highlight key actions and elements.
- Headline font: 'Playfair', a serif font, to add elegance. Body font: 'PT Sans', a sans-serif font for readability.
- Use simple, clear icons to represent file types and actions.
- Clean and organized layout with clear visual hierarchy.
- Subtle transitions for state changes and feedback.
