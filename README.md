# nextn

## Tech Stack

- Node.js
- React
- Next.js
- TypeScript
- Tailwind CSS

## Requirements

- Node.js 18+

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

## Commands

| Command | Script |
| --- | --- |
| `npm run dev` | `next dev --turbopack -p 9002` |
| `npm run start` | `next start` |
| `npm run build` | `NODE_ENV=production next build` |
| `npm run lint` | `next lint` |
| `npm run typecheck` | `tsc --noEmit` |

## Build

```bash
npm run build
```
